# 《混沌工坊》敌人设计 1.1v

本文档用于第一章敌人原型整理，基于《混沌工坊》核心数值设计与当前 Unity 敌人实现更新到 `1.1v`。文件名沿用 `v1`，正文版本以 `1.1v` 为准。

## 1. 设计目标

第一章敌人的职责不是单纯提高伤害，而是逐步教授玩家理解以下系统：

- 行动条节奏
- 混沌浪潮时机
- 距离与接敌窗口
- 免费移动的价值
- 三名角色在同一规则下的差异体验

敌人应当：

- 有明确意图与清晰反制点。
- 通过时间压力或空间压力制造选择。
- 尽量避免“纯数值秒杀式”威胁。

## 2. 敌人设计所依赖的基础常量

```text
行动条长度：12
混沌浪潮长度：12
混沌浪潮速度：1 格/秒
混沌浪潮与单位行动同刻时：先结算浪潮

玩家能量上限：3
玩家行动开始获得能量：1
混沌浪潮回能：恢复至 3

玩家初始抽牌：5
玩家行动开始抽牌：1
混沌浪潮抽牌：3
手牌上限：10

战场长度：20
每次行动免费移动：2
```

## 3. 教学节奏拆分

| 教学点 | 对应敌人 |
|---|---|
| 基础接敌与攻击范围 | 失控傀儡 |
| 高频行动与防御判断 | 锈刃追猎者 |
| 慢速重击与护盾处理 | 蒸汽盾卫 |
| 远程威胁与距离管理 | 枪械游民 |
| 行动条干扰与浪潮时机 | 混沌学徒 |

精英和 Boss 则继续放大这些考点：

- 断轴骑士：击退与力量反制
- 红线狙击手：远距离高压
- 熔炉执事：围绕混沌浪潮的护盾节奏
- 初代混沌炉心：把混沌浪潮机制做成整场 Boss 战的核心主题

## 4. 普通敌人

### 4.1 失控傀儡

```text
定位：教学近战
生命：28
速度：2
攻击范围：2
基础伤害：6
初始距离：7
```

核心体验：

- 让玩家理解“移动后再攻击”的基本流程。
- 明确建立攻击范围与接敌距离概念。

行为目标：

| 行为 | 条件 | 效果 |
|---|---|---|
| 蹒跚逼近 | 距离 > 2 | 向前移动 2 |
| 破损挥击 | 距离 <= 2 | 造成 6 点伤害 |
| 失衡重拍 | 连续两次因距离未命中后 | 下次命中造成 8 点伤害 |

原型结论：

- 这是最基础的节奏模板。
- 单体战斗中应允许玩家至少有 1-2 个舒适回合建立优势。

### 4.2 锈刃追猎者

```text
定位：高频压迫
生命：24
速度：4
攻击范围：1
基础伤害：4×2
初始距离：7
```

核心体验：

- 通过高行动频率压迫玩家处理防御、位移和抢节奏。
- 不是靠单次重击，而是靠“你不能忽视我”制造压力。

行为目标：

| 行为 | 条件 | 效果 |
|---|---|---|
| 贴地疾走 | 距离 > 1 | 向前移动 2-3 |
| 锈刃连斩 | 距离 <= 1 | 造成 4 点伤害，重复 2 次 |
| 追猎强化 | 玩家后撤后仍在中近距离 | 下次追击更快 |

角色对位：

| 角色 | 压力点 |
|---|---|
| 陆夕云 | 要用回身步与爆发窗口整理节奏 |
| 海洛安 | 最怕被连续多段磨血 |
| 拉缪 | 既能反制它，也可能在贴身对拼里吃亏 |

### 4.3 蒸汽盾卫

```text
定位：慢速护盾
生命：42
速度：1
攻击范围：2
基础伤害：11
初始距离：8
```

核心体验：

- 强调速度 1 敌人与混沌浪潮的节奏关系。
- 它慢，但每回合质量较高。

行为目标：

| 行为 | 条件 | 效果 |
|---|---|---|
| 举盾推进 | 距离 > 2 | 获得护盾并前进 |
| 蒸汽盾击 | 距离 <= 2 | 造成 11 点伤害 |
| 泄压防御 | 浪潮后的首次行动 | 获得额外护盾或强化推进 |

设计说明：

- 不是“高压秒人敌”，而是拖长战斗并迫使玩家思考爆发时机。
- 对海洛安来说，这是节奏镜像型对手。

### 4.4 枪械游民

```text
定位：远程压制
生命：26
速度：2
攻击范围：9
基础伤害：7
初始距离：10
```

核心体验：

- 测试玩家能否利用位移、突进或远程手段改变交战区间。
- 不应无限后撤到失去交互性。

行为目标：

| 行为 | 条件 | 效果 |
|---|---|---|
| 粗制射击 | 距离 4-9 | 造成 7 点伤害 |
| 惊惶后撤 | 距离 <= 3 | 后撤 3，并改变下一次射击质量 |
| 调整枪线 | 距离 > 9 | 向前移动寻找可射区间 |

角色对位：

| 角色 | 压力点 |
|---|---|
| 陆夕云 | 依赖疾步斩连续接敌 |
| 海洛安 | 需要关键突进牌，不然会被白白消耗 |
| 拉缪 | 可对射，但弹药续航会被检验 |

### 4.5 混沌学徒

```text
定位：行动条干扰
生命：34
速度：3
攻击范围：3
基础伤害：7
初始距离：8
```

核心体验：

- 让玩家真正注意行动条，而不仅仅是血量和距离。
- 伤害不高，但会扰乱节奏预期。

行为目标：

| 行为 | 条件 | 效果 |
|---|---|---|
| 扭曲秩序 | 玩家行动条接近出手 | 玩家行动条 -2 |
| 混沌短击 | 距离 <= 3 | 造成 7 点伤害 |
| 闪烁逼近 | 距离 > 3 | 向前移动 |
| 借潮 | 浪潮后的首次行动 | 自身行动条前推 |

限制原则：

- 同一学徒不应连续两次都只做行动条干扰。
- 威胁来自节奏变化，而不是无解控制。

## 5. 精英敌人

### 5.1 断轴骑士

```text
定位：反制击退的近战精英
生命：85
速度：2
攻击范围：3
基础伤害：12
初始距离：8
机制：被强制位移后获得力量
```

核心体验：

- 玩家仍可以击退它，但需要接受“击退换时间、也换来后续压力”的代价。

关键行为：

| 行为 | 条件 | 效果 |
|---|---|---|
| 轴步推进 | 距离 > 3 | 获得护盾并前进 |
| 骑士横扫 | 距离 <= 3 | 造成 12 点基础伤害 |
| 裂轴突进 | 力量足够且距离中近 | 消耗力量进行更强突进与重击 |
| 断轮守势 | 低血且无护盾 | 获得一次保命护盾 |

### 5.2 红线狙击手

```text
定位：远程精英
生命：70
速度：3
攻击范围：10
基础伤害：9
初始距离：10
机制：距离 6 以上伤害 +5
```

核心体验：

- 放任远距离会非常危险。
- 成功压近后，战斗难度应明显下降。

关键行为：

| 行为 | 条件 | 效果 |
|---|---|---|
| 红线瞄准 | 距离 >= 6 | 标记玩家 |
| 贯线射击 | 已标记且距离 >= 6 | 造成高伤害 |
| 腰射 | 距离 2-5 | 造成中等伤害 |
| 翻跃后撤 | 距离 <= 1 | 拉开距离并获得闪避 |

### 5.3 熔炉执事

```text
定位：浪潮护盾精英
生命：100
速度：1
攻击范围：3
基础伤害：14
初始距离：9
机制：每次浪潮获得 14 护盾
```

核心体验：

- 把混沌浪潮变成双方都在争夺的节拍点。
- 玩家需要考虑“在浪潮前打破盾”还是“在浪潮后重新组织爆发”。

关键行为：

| 行为 | 条件 | 效果 |
|---|---|---|
| 炉门合拢 | 混沌浪潮触发 | 获得 14 护盾 |
| 执事重锤 | 距离 <= 3 | 造成 14 点伤害 |
| 煤压推进 | 距离 > 3 | 获得小护盾并前进 |
| 炉温过载 | 护盾过高时 | 进一步提高攻击威胁 |

## 6. Boss：初代混沌炉心

```text
定位：第一章终局 Boss
生命：220
速度：2
攻击范围：3
初始距离：9
```

Boss 主题：

- 它应当是混沌浪潮规则的具象化敌人。
- 不是纯血厚伤高，而是不断改变节奏、距离和行动预期。

### 阶段 1：炉心苏醒

```text
生命区间：100%-70%
```

阶段目标：

- 给玩家观察 Boss 节奏的窗口。
- 建立“浪潮会让 Boss 变强”的心理预期。

关键行为：

- 近距离重击
- 中远距离稳定推进
- 浪潮后获得 10 护盾

### 阶段 2：牵引失控

```text
生命区间：70%-35%
阶段变化：速度 +1
```

阶段目标：

- 用更快的回合频率和拉近效果压缩玩家空间。
- 让位置管理变得更加紧迫。

关键行为：

- 攻击附带拉近
- 中距离牵引脉冲并加快自身节奏
- 对持续贴身的玩家给出反制

### 阶段 3：混沌爆燃

```text
生命区间：35%-0%
阶段变化：浪潮后随机改变距离并推进自身行动条
```

阶段目标：

- 把战斗推向更混乱但仍可读的终盘状态。
- 随机性要有压迫感，但不能完全失控。

关键行为：

- 近距离高伤敲击
- 中距离喷射攻击
- 远距离校准战场并改变距离

限制原则：

- 单次直接伤害不应长期超过 16。
- 连续随机改距的次数应有限，避免不可预判过头。
- 阶段 3 的压力主要来自节奏扭曲，而不该来自纯面板暴涨。

## 7. 遭遇建议

### 普通战

| 编号 | 组合 | 用途 |
|---|---|---|
| 1-1 | 失控傀儡 ×1 | 基础接敌教学 |
| 1-2 | 失控傀儡 ×1 + 锈刃追猎者 ×1 | 速度差异教学 |
| 1-3 | 枪械游民 ×1 | 远程压制教学 |
| 1-4 | 蒸汽盾卫 ×1 + 失控傀儡 ×1 | 护盾与目标优先级 |
| 1-5 | 混沌学徒 ×1 + 锈刃追猎者 ×1 | 节奏干扰与时间压力 |

### 精英战

| 编号 | 组合 | 用途 |
|---|---|---|
| E-1 | 断轴骑士 ×1 | 击退代价 |
| E-2 | 红线狙击手 ×1 + 失控傀儡 ×1 | 远程优先级 |
| E-3 | 熔炉执事 ×1 | 浪潮节奏对抗 |

## 8. 验收目标

| 战斗类型 | 时长目标 | 玩家平均掉血 |
|---|---:|---:|
| 普通战斗 | 35-60 秒 | 5-12 |
| 精英战斗 | 70-110 秒 | 15-25 |
| Boss 战 | 150-240 秒 | 25-45 |

额外检查：

- 普通敌的首次威胁应给玩家至少一次完整行动窗口。
- 高频敌人必须用较低单次伤害换取频率。
- 慢速精英要体现浪潮协同，而不是只靠数值硬顶。
- 至少有一类敌人对每名角色形成真实压力，但不应出现完全无解的角色克制。
