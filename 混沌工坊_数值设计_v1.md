# 《混沌工坊》数值设计 v1

本文档用于原型制作与脚本落地。核心目标是让行动条、混沌浪潮、距离、费用、抽牌和不同速度角色形成清晰差异。

## 1. 核心战斗规则

```text
行动条长度：12
单位行动条推进速度：每秒增加自身速度
单位行动条达到12时行动一次，行动后归零

混沌浪潮长度：12
混沌浪潮速度：1格/秒
混沌浪潮每12秒触发一次
```

当混沌浪潮与单位行动同时触发时：

```text
先结算混沌浪潮
再结算单位行动
```

速度1角色每次行动前吃满混沌浪潮，这是慢速重型角色的补偿和爽点。

| 速度 | 行动间隔 | 12秒内行动次数 | 定位 |
|---:|---:|---:|---|
| 1 | 12秒 | 1次 | 重型爆发 |
| 2 | 6秒 | 2次 | 慢速强攻 |
| 3 | 4秒 | 3次 | 标准节奏 |
| 4 | 3秒 | 4次 | 快速连动 |
| 5 | 2.4秒 | 5次 | 高频轻击 |
| 6 | 2秒 | 6次 | 临时爆发上限 |

## 2. 能量与手牌

```text
能量上限：3
单位行动开始时：获得1点能量
混沌浪潮触发时：所有单位能量恢复至3
行动结束后保留未使用能量

手牌上限：10
战斗开始抽牌：5
每次玩家行动开始：抽1张牌
混沌浪潮触发时：玩家抽3张牌
若抽牌后超过手牌上限，必须弃至10张
```

资源体验：

| 角色类型 | 行动频率 | 手牌压力 | 设计目的 |
|---|---:|---|---|
| 慢速角色 | 低 | 容易溢出 | 弃牌转收益，积蓄重击 |
| 标准角色 | 中 | 基本平衡 | 正常抽打循环 |
| 高速角色 | 高 | 容易缺牌 | 靠浪潮抽3补足消耗 |

费用定位：

| 费用 | 定位 |
|---:|---|
| 0费 | 位移、防御、资源调整、弃牌管理 |
| 1费 | 拼接动作、轻攻击、小收益 |
| 2费 | 主力攻击和战术牌 |
| 3费 | 完整行动的核心牌 |
| 4费以上 | 依赖预支、储能或特殊机制 |

## 3. 距离系统

```text
战场长度：20
玩家初始位置：5
敌人初始位置根据类型决定
每次行动可免费移动2格
免费移动不消耗能量，每次行动限1次
```

攻击命中条件：

```text
当前距离 <= 武器攻击范围
```

| 敌人类型 | 初始距离 |
|---|---:|
| 近战普通敌 | 7 |
| 标准敌 | 8 |
| 远程敌 | 10 |
| 精英 | 8-10 |
| Boss | 9 |

| 距离 | 状态 |
|---:|---|
| 0-1 | 贴身 |
| 2-3 | 近战 |
| 4-5 | 中距离 |
| 6-10 | 远程 |
| 11+ | 超远 |

## 5. 角色机制

### 陆夕云

```text
生命：48
速度：3
武器：长剑
核心：弱点攻击
```

弱点：

```text
弱点最多3层
攻击弱点目标时，每层弱点使伤害+25%
造成伤害后移除所有弱点
```

### 海洛安

```text
生命：60
速度：1
武器：战锤
核心：预支、罪债、赎罪、告解
```

预支：

```text
海洛安行动中可以预支最多3点能量
预支能让本次行动临时超过3点能量上限
每预支1点，获得1层罪债
```

罪债：

```text
罪债不会降低速度
罪债不会直接扣除能量
每层罪债使下一次获得的护盾或治疗效果-2
罪债最多5层
```

赎罪：

```text
赎罪效果可以移除罪债，并获得额外收益
如果没有罪债，赎罪牌仍有基础效果，但收益较低
```

告解：

```text
每弃置1张牌，获得1层告解，最多5层
下一张战锤攻击牌或护盾牌消耗全部告解
每层告解使该牌伤害或护盾+2
若弃牌发生在混沌浪潮后的手牌溢出，每弃1张额外获得1点护盾
```

设计目的：海洛安行动少、消耗牌慢，容易因浪潮抽3导致手牌溢出。弃牌不作为纯惩罚，而是转化为下一次重击或壁垒的蓄力。

### 拉缪

```text
生命：42
速度：5
武器：短刀、手弩
核心：远近切换、弹药、位移爆发
```

双武器：

```text
短刀牌偏近战、多段、流血
手弩牌可远程攻击，但消耗弹药
从距离4以上突进到距离1以内时，下一张短刀牌伤害+2
```

## 7. 卡牌数值公式

```text
0费攻击：武器伤害 × 0.4
1费攻击：武器伤害 × 1.0
2费攻击：武器伤害 × 1.6
3费攻击：武器伤害 × 2.2
4费攻击：武器伤害 × 2.8-3.0
```

| 附加效果 | 伤害折价 |
|---|---:|
| 移动2 | -10% |
| 移动3 | -20% |
| 移动4 | -30% |
| 击退/拉近2 | -20% |
| 抽1张牌 | -25% |
| 施加弱点/易伤 | -25% |
| 获得护盾 | -20%到-40% |

## 8. 混沌浪潮效果

混沌浪潮每12秒触发：

```text
所有单位能量恢复至3
玩家抽3张牌
触发1个随机浪潮效果
```

随机池比例：

```text
正面：35%
中性：40%
负面：25%
```

| 效果 | 类型 |
|---|---|
| 所有单位行动条+2 | 中性 |
| 所有单位速度+1，持续6秒 | 中性 |
| 玩家抽2张牌 | 正面 |
| 玩家恢复3点生命 | 正面 |
| 敌人行动条-2 | 正面 |
| 双方距离随机变化2 | 中性 |
| 所有攻击范围-1，持续6秒 | 中性 |
| 玩家当前能量-1 | 负面 |
| 最近敌人获得8护盾 | 负面 |
| 所有单位被拉近2 | 中性偏危险 |

## 10. 原型验收目标

```text
普通战斗：35-60秒
精英战斗：70-110秒
Boss战：150-240秒

普通战斗平均掉血：5-12
精英战斗平均掉血：15-25
Boss战平均掉血：25-45
```

角色体验目标：

| 角色 | 核心体验 |
|---|---|
| 陆夕云 | 稳定接敌，制造弱点，抓窗口爆发 |
| 海洛安 | 吃满浪潮重拍，预支神罚，弃牌告解，再通过赎罪消化罪债 |
| 拉缪 | 高频行动，远近切换，用移动和弹药消耗大量手牌 |
